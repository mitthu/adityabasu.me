## Manaully add ssh keys to the agent

Add identities -
```bash
# On Linux
ssh-add -k

# On macOS
ssh-add -K
```

List loaded identities

```bash
ssh-add -l
```

## Automatically load keys

To auto-load to the agent while logging in -

```bash
echo "AddKeysToAgent yes" >>~/.ssh/config
```
